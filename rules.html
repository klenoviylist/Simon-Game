Rules

1. Press any key on your keyboard to start the game.

2. Simon will give the first signal. Repeat the signal by clicking the same color button.

3. Simon will duplicate the first signal and add one. Repeat these two signals by clicking the same color buttons, in order.

4. Simon will add a color as long as you repeat each sequence of signals correctly.

5. If you fail to repeat the sequence of signals correctly, game ends. You can start a new game by pressing any key on your keyboard.
